# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer built using **Python** and **Flask** to help users evaluate their resumes, calculate ATS scores, detect skills, and receive intelligent suggestions for improvement.

## 🚀 Features (Planned)

* 📄 Upload Resume (PDF/DOCX)
* 📊 ATS Score Calculation
* 🧠 Skill Detection
* 💼 Job Description Matching
* 🤖 AI-Powered Resume Suggestions
* 📈 Resume Improvement Tips

## 🛠️ Technologies Used

* Python
* Flask
* HTML5
* CSS3
* spaCy
* pdfplumber
* python-docx

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/OmprakashRoutray/AI-Resume-Analyzer.git
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:

   ```bash
   python app.py
   ```

4. Open your browser and visit:

   ```
   http://127.0.0.1:5000
   ```
